# DBMS & SQL

This folder contains resources on Database Management Systems and SQL for placement preparation.

## Topics to Cover

### DBMS Fundamentals
- ER Diagrams & Entity Relationship Model
- Normalization (1NF, 2NF, 3NF, BCNF)
- Functional Dependencies
- Relational Algebra
- Transaction Management (ACID properties)
- Concurrency Control
- Indexing & B-Trees
- File Organization

### SQL
- DDL: CREATE, ALTER, DROP, TRUNCATE
- DML: SELECT, INSERT, UPDATE, DELETE
- Joins: INNER, LEFT, RIGHT, FULL OUTER, CROSS, SELF
- Aggregation: GROUP BY, HAVING, ORDER BY
- Subqueries & Nested Queries
- Views, Triggers, Stored Procedures
- Window Functions
- Query Optimization

### Interview Focus Areas
- Difference between DELETE, TRUNCATE, DROP
- ACID vs BASE
- CAP Theorem in Databases
- SQL vs NoSQL
- Indexing and Query Performance
- Joins and their use cases
