# DSA — Data Structures & Algorithms

This folder contains resources on Data Structures and Algorithms for placement preparation.

## Topics to Cover

### Data Structures
- Arrays & Strings
- Linked Lists (Singly, Doubly, Circular)
- Stacks & Queues
- Trees (Binary Tree, BST, AVL, Red-Black)
- Heaps & Priority Queues
- Hash Maps & Hash Sets
- Graphs (Directed, Undirected, Weighted)
- Tries
- Segment Trees & Fenwick Trees
- Disjoint Set Union (DSU)

### Algorithms
- Sorting: Bubble, Selection, Insertion, Merge, Quick, Heap, Counting, Radix
- Searching: Binary Search, Ternary Search
- Recursion & Backtracking
- Dynamic Programming (Memoization & Tabulation)
- Greedy Algorithms
- Graph Algorithms: BFS, DFS, Dijkstra, Bellman-Ford, Floyd-Warshall, Kruskal, Prim
- Sliding Window & Two Pointers
- Divide and Conquer
- Bit Manipulation

### Complexity Analysis
- Time Complexity
- Space Complexity
- Big O, Omega, Theta Notation
