# Operating System

This folder contains resources on Operating Systems for placement preparation.

## Topics to Cover

### OS Fundamentals
- What is an Operating System?
- Types of OS: Batch, Time-sharing, Distributed, Real-time, Embedded
- Functions of OS: Process management, Memory management, File system, I/O management
- Kernel vs User Space
- System Calls

### Process Management
- Process vs Thread
- Process States: New, Ready, Running, Waiting, Terminated
- Process Control Block (PCB)
- Context Switching
- Multithreading & Concurrency
- Inter-Process Communication (IPC): Pipes, Shared Memory, Message Passing, Semaphores, Sockets

### CPU Scheduling
- FCFS (First Come First Serve)
- SJF (Shortest Job First)
- SRTF (Shortest Remaining Time First)
- Priority Scheduling
- Round Robin
- Multilevel Queue Scheduling
- Preemptive vs Non-preemptive

### Synchronization & Deadlock
- Race Condition
- Critical Section Problem
- Mutex vs Semaphore
- Peterson's Algorithm
- Banker's Algorithm
- Deadlock: Conditions, Prevention, Avoidance, Detection, Recovery

### Memory Management
- Logical vs Physical Address
- Contiguous Allocation: Fixed & Variable Partition
- Paging & Page Table
- Segmentation
- Virtual Memory
- Demand Paging & Page Faults
- Page Replacement Algorithms: FIFO, Optimal, LRU, LFU
- Thrashing

### File System
- File Attributes, Operations, Types
- Directory Structure: Single-level, Two-level, Tree, Acyclic Graph, General Graph
- File Allocation Methods: Contiguous, Linked, Indexed
- Free Space Management: Bit Vector, Linked List, Grouping, Counting

### Disk Management
- Disk Scheduling: FCFS, SSTF, SCAN, C-SCAN, LOOK, C-LOOK
- RAID (0, 1, 5, 10)
